# 色彩

## 主色

整体色彩采用中国传统古色，取色来自 b 站`@设计师IORI`，组件库灵感来自`@LAINE001/ikun-ui`

其颜色分别作为`默认` `主题` `成功` `警告` `信息` `危险`这些行为的主色
